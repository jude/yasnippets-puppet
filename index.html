yasnippets-puppet by jude

yasnippets for puppet-mode

This is my collection of yasnippet snippets for puppet-mode. The snippets include context-sensitive expansions for - for various puppet resource types

Dependencies

yassnippet >= 0.6.1c emacs >= 23.3 (Based on my current environment. Earlier versions may work.)

Install

Assuming your yasnippet snippets directory is at ~/.emacs.d/snippets: > mkdir -p ~/.emacs.d/snippets/text-mode/puppet-mode >
