yasnippets-puppet

yasnippets snippets for puppet-mode

This is a collection of yasnippet snippets for puppet-mode. The snippets feature context-sensitive - expansions for various puppet resource types.

Dependencies

  • yassnippet >= 0.6.1c
  • emacs >= 23.3
(Based on my current environment. Earlier versions may work.)

Install

Clone the repo, or extract the tarball and create a link from the repos puppet-mode directory to the directory yasnippet expects the puppet-mode snippets:

> git clone git@github.com:jude/yasnippets-puppet.git /your/working/directory
> ln -s /your/working/directory/puppet-mode ~/.emacs.d/snippets/text-mode/puppet-mode
